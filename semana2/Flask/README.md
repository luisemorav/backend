## Crear un entorno virtual
```
python -m venv venv
```

## Activar el entorno virtual
```
- Windows venv/Scripts/activate
- MAC: source venv/bin/activate
```

## Instalar Flask
Importante: Tener el entorno activo!!!

```
pip install Flask
```

## Desactivar el entorno al final
```
deactivate
```
